## Core Concepts

### Epistemic Shock Index (ESI)

```
ESI = (1 - Integrity) √ó (1 - ACC) √ó (1 + Instability)
```

### Chain of Custody (ACC)

Complete testimony requires 5 elements:

1. **Source Entity** - Who/What is witnessed? (ID established)
2. **Timestamp** - When did it occur? (ISO 8601 format)
3. **Witness Identity** - Who was present to observe?
4. **Method/Log** - What action was taken? (Documentation)
5. **Outcome Link** - Can the final result be traced? ‚ö†Ô∏è **CRITICAL** - Most common failure point

**Accountability Score Calculation:**

```
ACC = (Elements Present) / 5
```

**Target:** ACC ‚â• 0.90 for testimonial completion
